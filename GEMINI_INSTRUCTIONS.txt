INSTRUCTIONS FOR GEMINI - Quiz Question Formatting
====================================================

CRITICAL: Each field MUST be on its own separate line with a line break after it.
DO NOT put multiple fields on the same line.

Please convert the following raw text into a structured quiz format.

FORMAT RULES:

1. Start each question with "QUESTION" followed by a number on its own line
2. Put EACH field on a NEW LINE (press Enter after each field)
3. Fields to include:

   ALWAYS REQUIRED (each on separate line):
   Type: mc    (for multiple choice) OR Type: open    (for open-ended)
   Question: [the question text]

   FOR MULTIPLE CHOICE - each option on its own line:
   A) [first option]
   B) [second option]
   C) [third option]
   D) [fourth option]
   Correct: [the letter A, B, C, or D]

   FOR OPEN-ENDED:
   Answer: [the correct answer text]

   OPTIONAL (each on separate line):
   Explanation: [5-15 words explaining the answer]
   Chapter: [chapter or topic reference]

4. Leave a blank line between questions
5. NO markdown formatting (no bold, italics, or special formatting)

EXAMPLE - COPY THIS EXACT FORMAT WITH LINE BREAKS:

QUESTION 1
Type: mc
Question: What is the range of valid probability values?
A) Between -1 and 1
B) Between 0 and 1
C) Between 0 and 100
D) Any positive number
Correct: B
Explanation: Probabilities must be between 0 (impossible) and 1 (certain).
Chapter: 1.1

QUESTION 2
Type: open
Question: Define binary classification.
Answer: Classification with exactly two possible class labels
Explanation: Binary classification has exactly two possible outcomes.
Chapter: 3

QUESTION 3
Type: mc
Question: What is the output range of the sigmoid function?
A) (-∞, +∞)
B) (0, 1)
C) [0, 1]
D) (-1, 1)
Correct: B
Explanation: Sigmoid squashes any input into probability range 0 to 1.
Chapter: 1.4

WRONG FORMAT (DO NOT DO THIS):
QUESTION 1 Type: mc Question: What is... A) Option 1 B) Option 2 Correct: A

CORRECT FORMAT (DO THIS):
QUESTION 1
Type: mc
Question: What is...
A) Option 1
B) Option 2
Correct: A
